# Smart Task Manager

A desktop task management application built with Python and Tkinter. Users can create, organize, search, complete, and delete tasks with automatic JSON-based persistence.

## Features

- Add new tasks
- Complete tasks
- Delete tasks
- Search tasks by keyword
- Filter by:
  - All Tasks
  - Active Tasks
  - Completed Tasks
- Automatic JSON persistence
- Clean desktop interface

## Technologies

- Python 3
- Tkinter
- JSON
- Git
- GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/RobJorge78/smart-task-manager.git
```

Move into the project:

```bash
cd smart-task-manager
```

Run:

```bash
py main.py
```
